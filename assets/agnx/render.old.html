智能体产品生态系统

AgnFlow、AgnChat、AgnAuto、AgnTest四大产品共同构建了完整的智能体开发生态系统，覆盖工作流、人机交互、自动化测试等关键领域

AgnFlow

智能体工作流基础框架，快速开发复杂智能体工作流

核心特性

  • 智谱免费API集成
  • 多智能体通信协议：a2a, mcp
  • 函数调用路由机制
  • 支持RabbitMQ/Kafka消息队列
  • JSONRPC/HTTP/SSE协议支持
  • FastAPI集成
  • MediaCrawler集成
FastAPI RabbitMQ JSONRPC

AgnChat

智能体聊天WebUI应用，支持多模态交互与动态代码生成

核心特性

  • 多模态交互支持（文本、语音、图像）
  • 智谱AI深度集成
  • 人机交互工作流：复述-解析-编辑-执行
  • XML指令解析与确认机制
  • 动态执行生成代码（AIPY）
  • 智能体进化（OpenEvolve）
  • 分布式部署架构
多模态 XML解析 分布式
